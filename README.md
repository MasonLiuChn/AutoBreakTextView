# AutoBreakTextView

解决android系统TextView自动换行不美观问题

# Feature

Android系统中的TextView会在行尾是连串的数字、字母或者标点符号时提前换行，但有些情况我们想要自然换行，就是排到一行末尾再换行而不是根据字母断开换行。

Usage
=====

```java
repositories {
    maven {
        url "https://jitpack.io"
    }
}

dependencies {
    compile 'com.github.MasonLiuChn:AutoBreakTextView:1.0.0'
}
```
